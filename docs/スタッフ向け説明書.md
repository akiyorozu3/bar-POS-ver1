# バーPOS スタッフ用 かんたん説明書

注文・会計・打刻のやり方をまとめた、スタッフさん向けの説明書です。
むずかしい操作はありません。この通りにやれば大丈夫です 🙆

---

## ⭐ 知っておくと安心なこと

- 入力した注文は**自動で保存**されます。うっかりリロードしても、アプリを閉じても、注文は残っています（同じ卓を選べば表示されます）。
- 別のiPadやPCで開いても、**同じ卓・同じ注文**が見えます。
- ネット接続は必要です。電波が悪いと保存・同期に失敗することがあるので注意。

---

## 1. ログイン

1. iPad で アプリを開く
2. **ユーザーID**：`staff` と入力
3. **パスワード**：お店で教わったものを入力
4. 「ログイン」を押す

> 売上やキャストのバックは、スタッフのアカウントでは見られません（オーナー専用です）。

---

## 2. 打刻（出勤・退勤）

出勤したとき・帰るときに、画面右上の「**打刻**」ボタンから記録します。

1. 右上の「**打刻**」を押す
2. **自分（キャスト）をプルダウンで選ぶ**
3. 出勤時は「**出勤**」、帰るときは「**退勤**」を押す → その時刻で記録されます
4. 下に**今日の打刻一覧**が出るので、押せたか確認

> 押し忘れ・押し間違いがあっても、オーナーが後から直せます。気づいたらオーナーに一言伝えてください。

> 🕔 営業日は「夕方17:00〜翌17:00」で1日です。なので**深夜（翌0時〜夕方まで）の退勤は“前日”の勤務**として記録されます（例：7/4の深夜2時の退勤 → 7/3の勤務）。おかしく見えても正常です。

---

## 3. 注文を入力する（「注文入力」画面）

### ① 席（卓）を開く／選ぶ
最初は席が空の状態です。お客様が入ったら卓を開きます。
- **「＋追加」** で新しい卓を開く。
- または上に並ぶ**定型ボタン（C1・C2… などよく使う卓）**をタップするとその卓を開けます。
- すでに開いている卓は、上のチップをタップして選びます。
- 席の名前（例：田中様）を入れられます。一人のお客様なら「**一人客**」をタップ。
- 卓を閉じたいときは、その卓の「**✕**」。（会計が終わった卓は**自動で消えます**）

### ② 卓の担当を選ぶ
伝票エリアの「**卓の担当**」で、その卓を担当するキャストを選びます。
- 担当が**2人以上**いるときは「**＋担当**」を押して人数分追加します（卓バックは担当で頭割りになります）。
- 担当を外すときは各行の「**×**」。

> 💡 卓の担当は必ず選んでください。ここに「卓バック」が入ります。未設定だとバックが付きません。

### ③ メニューを選ぶ
左側のメニューから商品をタップすると、右の伝票に追加されます。
- 上のタブで「本日限定」「セット」「ショット」「シャンパン」「キャストドリンク」…とカテゴリを切替（タブは横にスワイプ）。
- 同じ商品をもう一度タップすると数量が増えます。
- 伝票の「**−／＋**」で数量を調整、0 にすると消えます。

### ④ キャストドリンクの担当を選ぶ
**キャストドリンク**を伝票に入れると、その行に「**ドリンク担当**」の選択が出ます。**そのドリンクをもらった子**を選んでください。
- ここで選んだ子に「ドリンクバック」が付きます（卓の担当とは別の子でもOK）。
- 例：卓の担当はたか子、でも「キャストドリンク（ゆう子）」ならドリンク担当に**ゆう子**を選ぶ。
- キャストドリンク以外の商品には担当を選ぶ必要はありません。

### ⑤ フリー入力・割引（メニューに無いもの）
「フリー入力」タブで、ボトルチャージ・指名料・延長料などや、**割引**を入れられます。
- よく使うものはボタン（プリセット）をタップするだけ。
- それ以外は「品名」と「金額」を入れて「追加」（金額が税込か税抜かはお店の設定によります）。
- **割引**は割引の行から入れます（合計から引かれます）。

---

## 4. お会計（「会計へ」→「会計」画面）

1. 伝票の下の「**会計へ →**」を押す（または上部「会計」タブ。会計タブはいつでも開けます）
2. 会計画面の**上に「未会計の卓」がチップで並ぶ**ので、会計したい卓をタップ。注文内容と合計金額が出ます
3. **支払い方法**を選ぶ：**現金 / カード / QR払い**

### 現金のとき
- **お預かり金額を入力**します（ぴったり額や 1,000・5,000 円などのボタン、または金額を直接入力）
- **お釣り**が自動で表示されます

### カード / QR のとき
- 金額を確認するだけ（手数料などの詳細はオーナー画面で管理されます）

4. 最後に「**支払い確定**」を押す → 完了！
   伝票は保存され、**その卓は自動で消えます**。

> 金額を間違えたまま「支払い確定」しないよう注意。確定後の取り消しはオーナー対応になります。

---

## 5. 表示金額について

お店の消費税設定によって表示が変わります（どちらでも、合計欄の金額がお客様の支払額です）。
- **税込で登録している場合**: メニューの価格がそのままお客様の支払額。消費税の行は出ません。
- **税抜で登録している場合**: 明細は税抜で並び、最後に「消費税」と「合計（税込）」が出ます。

迷ったら、**「合計」欄の金額**をお客様に伝えればOKです。

## 6. 困ったとき

| こんなとき | どうする |
|---|---|
| ログインできない | ID `staff` とパスワードを確認（半角で入力） |
| メニューが出てこない | ネット接続を確認。オーナーに連絡 |
| 卓が見当たらない | 「＋追加」や定型ボタンで開く。別の卓を選んでないかも確認 |
| 打刻を押し忘れた・間違えた | オーナーに伝える（オーナーが直せます） |
| 金額や担当を間違えた（会計前） | 伝票で数量・担当を直す。商品は−で消せる |
| 会計を間違えて確定した | オーナーに連絡（スタッフでは取り消せません） |
| 「締め済み」で会計できない | その日はオーナーがレジ締め済み。オーナーに連絡 |
| 画面上の日付が今日じゃない/オレンジ | 「今日に戻す」を押す（分からなければオーナーに確認） |
| 変な卓（席A/B/Cなど）が出る・消えない | iPad を一度開き直す（タブを閉じて開く）。分からなければオーナーへ |

---

困ったら、まずオーナーさんに聞いてください。
